# NativeApp

## 由于浏览器安全限制请遵循以下操作：<br>     
### 1、不要使用任何版本的IE浏览器，尽量使用版本最新的现代浏览器。（如火狐，谷歌，360浏览器等）<br> 
### 2、请将要读取的文件先保存在当前的file文件夹内，然后再添加。<br>
### 3、如果打开文件失败，请尝试先打开浏览器再打开index.html，或者检查当前的file文件夹内是否有此文件。<br>
### 4、如果手动清除浏览器缓存将会清除之前保存的所有记录。<br>
### 5、参考帮助页面的文档。<br>
